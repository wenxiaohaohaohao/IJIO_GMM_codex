## Phase M0：中间品链路审计（前置）

### 背景与目的

二阶段 GMM 框架中，`m`（中间品投入）是 generated regressor，来自一阶段估计。
- **风险**：一阶段与二阶段口径错配 → 矩条件失效 → `b_m` 不稳定、符号失真
- **解决**：先锁定口径、核查兼容性、运行诊断，确保后续估计有坚实基础

#### 尹恒替代口径定义

**文件来源**：
- 变量字典：`tmp_yinheng_var_dict.docx`
- Gauss代码：`reference_materials/学习材料/yinheng_2024/程序/Gauss/GMM_ind*.prg`（L182-186）

**尹恒 `m` 生成链**：
```
em = ln((imaterial + d23) / 1000)    // 全口径可变成本（含进项税），名义值
pm = ln(inputprice_s)                 // 行业级投入价格指数
m  = em - pm                          // 实际中间品 = 名义 - 价格指数对数
```

**口径对照表**：

| 维度 | 主代码口径 | 尹恒口径 |
|------|----------|---------|
| 中间品范围 | 仅国内中间投入 (`delfateddomestic`) | 全口径可变成本含进项税 (`imaterial + d23`) |
| 平减方式 | 预平减（嵌入变量中） | 实时平减（em - pm, 行业级投入价格指数） |
| 平减指数 | 不详（嵌入数据预处理） | `inputprice_s`（统计局+IO表） |
| 资本 | Brandt-Rawski 永续盘存法 | 固定折旧率9%平减 |
| 产出 | Q = Q1/outputdef2 | revenue = d1+d24（含销项税） |
| 劳动 | 全部从业人员 | 全部从业人员（一致） |

#### **M0.5：诊断回归脚本**

**目标**  
运行对照实验，检验口径/样本错配是否导致 `b_m` 失稳。

**方法**

固定 IV、硬联立结构不动，运行两组估计：
1. **当前样本**下估计（baseline）
2. **严格同样本**下估计（两阶段都用交集样本）

比较输出：
- `b_m` 值与符号  
- `J_p` 值  
- `gmm_conv` flag  
- 资本/劳动总弹性

**脚本要求**
- `RUN_POINT_ONLY=1`, `RUN_BOOT=0`, `RUN_DIAG=1`
- 先 G1 后 G2
- 将两组结果并排输出到日志

**产出**
- 诊断日志：两组结果对比  
- 初步判定：口径/样本问题占比多少

**通过标准**  
✓ 若同样本后 `b_m` 收敛，优先判定为口径/样本错配问题

**执行者**：Codex
