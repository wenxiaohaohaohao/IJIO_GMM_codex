# IJIO_GMM_codex_en

本仓库用于 IJIO 相关实证研究的代码、结果与资料管理，目标是保持可复现、可追踪、可快速迭代。

## 目录概览

- `1017/`：主研究代码与阶段性结果
- `workflow/`：通用工作流脚本与模板
- `图表/`：图表与导出结果
- `文章参考/`：参考文献与外部材料（只读）
- `备份/`：历史备份内容（只读）
- `path_rules.md`：路径规则说明
- `权限管理说明.md`：实时权限规则（当前有效）
- `AGENTS.md`：Codex 协作与执行规则

│  ├─ src/                               [可删改替换]

├─ 备份/                                  [只读可调用]

├─ path_rules.md                         [规则文档，可编辑]

属性判定规则（Codex 执行时使用）：
- 标记为 `只读可调用`：只允许读取和调用，不允许修改、replace、删除、重命名、移动。
- 标记为 `可删改替换`：允许新增、修改、replace、删除、重命名、移动。
- 标记为 `规则文档/说明文档，可编辑`：按普通可编辑文件处理。

## 快速开始

1. 使用 PowerShell 进入仓库根目录：`D:\paper\IJIO_GMM_codex_en`
2. 根据任务进入对应目录（如 `1017/1022_non_hicks/code/`）
3. 运行对应脚本并将输出写入非只读目录
4. 变更后用 `git status` 检查工作区状态

## 权限约定（摘要）

- 只读可调用：`文章参考/`、`备份/`
- 可删改替换：除以上两目录外的其余工作区文件

完整规则见：`权限管理说明.md`

## 维护建议

- 临时脚本统一使用 `tmp_` 前缀，任务完成后及时清理
- 结果文件尽量写入固定结果目录，避免散落在根目录
- 提交前优先检查是否误改只读目录
